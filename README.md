# My TokyoNight Dotfiles

**Hyprland, Waybar, SwayNC and more!**

Inspired by [Bread On Penguins'](https://github.com/BreadOnPenguins/dots) and [stronk-dev's](https://github.com/stronk-dev/Tokyo-Night-Linux) dotfiles. You can also find some of my bash scripts in the `.bin` folder.

# GTK Theme

The GTK theme used is [Tokyo-Night-GTK](https://github.com/Fausto-Korpsvart/Tokyonight-GTK-Theme), and in my system its set with `nwg-look`

# Programs needed

There are required programs, but some of them won't be needed if you remove their respective keybindings in the config files.

## Required

- Hyprland
- Hyprpaper
- SwayNC
- Neovim
- Waybar
- Spotify with `spicetify`
- Rofi
- Kitty
- ZSH

## Optional (but recommended)

- OpenRGB
- KDE Connect
- PCManFM
- LibreWolf
- Vesktop
