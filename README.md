# My Multiple-Themed dotfiles

**Hyprland, Waybar, SwayNC and more!**

Inspired by [Bread On Penguins'](https://github.com/BreadOnPenguins/dots) and [stronk-dev's](https://github.com/stronk-dev/Tokyo-Night-Linux) dotfiles. You can also find some of my bash scripts in the `.bin` folder.

Use the command `themeset <theme-id>` to change the system theme.
**Available themes:**

- Tokyo Night (id: `tokyonight`)
- Gruvbox (id: `gruvbox`)
- Catppuccin Latte (id: `catppuccin-latte`)

## GTK Themes

The GTK themes used are:

- [Tokyo-Night-GTK](https://github.com/Fausto-Korpsvart/Tokyonight-GTK-Theme) / `tokyonight`
- [Gruvbox-GTK](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme) / `gruvbox`
- [Catppuccin-GTK Latte](https://github.com/Fausto-Korpsvart/Catppuccin-GTK-Theme) / `catppuccin-latte`

GTK themes must be installed in `~/.themes` and the folder name should be the `theme-id`.

## Icons

The icons are obtained from [fkorpsvart's](https://www.pling.com/u/fkorpsvart) collection on Pling. They should follow the same rule as themes, but have to be placed in ~/.icons.

## Programs needed

There are required programs, but some of them won't be needed if you remove their respective keybindings in the config files.

### Required

- Hyprland
- Hyprpaper
- SwayNC
- Neovim
- Waybar
- Rofi
- Kitty
- ZSH

### Optional (but recommended)

- OpenRGB
- KDE Connect
- PCManFM
- LibreWolf
- Vesktop
